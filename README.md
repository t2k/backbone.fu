##Config/dependencies:
#### install coffee-script and bower globally
1. npm install -g coffee-script bower

### install client side libraries via bower.json
2. bower install

3. npm install

### coffeescript: compile and watch
4. coffee -o js/ -cw js/
5. coffee -o lib/ -cw src/
